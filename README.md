# 代码管理仓库
## 仓库的作用
* a. 社区&网站代码协作平台
* b. 文章&代码&项目开发协作平台
## 如何参与增加和维护代码
* 每位同学将自己的代码上传到GitHub平台，大家需积极提交Issues 和 Patch，亦接收外界的Issues 和 Patch。
* 提交代码时要附带代码说明文档
* 团队项目可以在GitHub上协作完成，先构建闭源仓库，待项目完善后再进行开源。
